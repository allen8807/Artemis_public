#  惟留风月        
  
## 惟留风月。尾声  
  
　　我合上本破旧的《论恩》，抿一口莲心茶，望向窗外的浮云。  
　　午后的阳光懒懒的，茶馆也似午睡了般的恬静。  
　　“以师傅的道行，这些上清宗的典籍想是早已烂熟于心了罢？”世远将他手中的书卷翻过一页，抬头问道，“虽知师傅怀疑青妍公主便是上清宗的师祖，没想竟特意去寻了来，还细读了这一下午。”  
　　我笑笑，“有时，即便是早已烂熟的东西，静下心来，回头再看一看，又会有些新的感慨。”  
　　“这本书，去年在斐城谦王府时曾翻过。记得讲的是人生来便承了世间各式各样的恩泽。而之后那一本，”世远指指我右手边的那本《论偿》，“讲的是人该怎么样去将这些恩泽偿还给世间，以求此生圆满。”  
　　我点点头，这孩子果然是聪慧过人，看过一遍的书竟总结得如此好。  
　　“然我现在所看的这本古人推崇的《说道》，却又强调凡事顺其自然，讲究个‘无为’。这些书看似都很有道理，却又相互矛盾。看一本便罢，自是有可取之处，可若是都看了，岂不又等于什么都没看？”  
　　我一手撑着腮，看着世远——这可真是个极难回答的问题啊。  
　　“对我来说罢……知晓这些就是知晓他人做事的缘由，多了份理解才好去其糟粕取其精华；多了份理解，许多烦扰也就化为了宽恕——总比想不明白要好些。这些学说，说到底也就是个角度的区别。每个人有天生重视的事、偏爱的角度，所以他们觉得正确的东西也便看似不同了。对你自己来说，自己做事的时候，选一个觉着最好的便是了。只是看别人的时候，别只想着你自己的原则就是。”  
　　世远乖巧地点点头。  
　　说到角度问题，我又想起些什么，决定趁这个机会给这徒儿上一课。便问世远道：“若有人对你说：‘人生来便有罪，人的一生是不断地赎罪。越是身世好的人，所带的罪过便越大。’你会如何想？”  
　　“身世好不好，是天定的，与个人的罪过何干？”世远不假思索地回答道。  
　　“那么你先放开身份这一说，单看那句‘人身来便有罪’呢？”  
　　世远听出我是想让他思考这句话的正确之处，便撑着脑袋思索了起来，半晌若有所思地答道，“若是说人生来就得到了些什么，这意味着有别的什么人失去了什么的话。若这使得别人产生了痛苦，倒也的确可以称之为罪过。”  
　　“所以，现在你再回头看看那句‘身世越好，罪过便越大’呢？”  
　　“……若想到这世间之物并非取之不尽，那么一个人得到的越多，便会有人失去更多，或是有更多人失去了什么，那么与他人的距离也就越大，也就是说无形中使得更多的人痛苦了。于是……他的罪便越大？”世远试探性地问道。  
　　“正是如此，”我点点头，“这便是五百年前，沧烬邪教的教义。”  
　　世远没有说话，只是惊讶地望着我。  
　　“其实，即便是‘邪教’，若说不出个道理来，又怎会有那么多的人信呢？”我笑着摇摇头，“你再顺着他们的思路想想，‘受苦的人要让有罪的人付出代价’便讲得通了罢？那么什么是代价呢？什么是赎罪呢？死，而且是痛苦地死亡。听起来依然很有道理罢？因为无数的‘受苦的人’因他们的罪而死去，同样的，他们需要偿还的便是生命。”  
　　“虽总觉得哪里不对，但似乎的确无可辩驳。”世远说着，皱了皱眉头，似乎想要找出这些话的破绽。  
　　“其实，他们的错，并非他们所信奉的教义本身，而是他们以为这教义乃是唯一的‘正确’。若你不跳出这个圈子，仅限在它内部去思考的话，便很容易被它所左右，特别是当你身边的所有人都如此告诉你的话，你便很容易认为这便是‘真理’，因为它的的确确是对的。可是，正确的，只有一个吗？当时的青妍，也曾因无法否认这样的教义，而迷惘过。她曾认为生在皇家，便承载着万民的期待，所以她也一直为了回应这样的期待而努力着。可是被囚于庸州之后，她的信念却一度被击垮——她发现民对她的并非是期待，而是怨恨，她开始怀疑，生在皇家，真的是罪吗？而她真的要以那些沧烬教徒所说的方式来赎罪吗？”  
　　世远看着我，等着我继续说下去。  
　　“那么。你方才说：‘人生来就得到了些什么，这意味着有别的什么人失去了什么’。你现在把这句话反过来说说看？”  
　　“……有别人失去了什么，才会有人得到了什么？”  
　　“明白了吗？”我笑道。  
　　世远深思了一会，恍然大悟道，“原来这便是上清宗‘人生来承恩’的由来吗？因为别人无形中的失去，我才会有所得，这便是他人予我的恩……没想到帝君与沧烬所说的道理，看似截然相反，却又是一体！”  
　　我一边笑着揉了揉世远的脑袋，一边回忆起了我在梦中所见到的青妍——从自以为在回应的他人期待的天真，到发现人与人之间的隔阂后的迷惘，再到无关他人而只做自己应做之事的豁达——截然不同的看法，只因一念之差。  
　　“不过这世间道理竟能因众人的不同而变幻出这万千的姿态，倒也真是奇妙。”世远感叹道，“青妍公主的她竟能跳出那个圈子，将事情反过来想一遍，这便是人们所说的‘悟’罢？”  
　　“那时候啊，有个同样身在皇家的人对她说了堆大话呢。”我回忆着重霄在赏青崖上的话，心里有微微的得意。  
　　“重霄帝君？难怪说上清宗的教义乃是帝君的旨意……”世远点点头，“话说回来，单是关于重霄帝君，却也能衍生出那么多教派来。这里的云天宗便又和颢宇上清宗的说法有所不同……”  
　　想到云天宗与上清宗的分歧，我不由地叹了口气——  
　　我如今身处的慕国，算是重霄信仰最早的发源地。而云天宗所奉的，便是慕国之国教。  
　　当年昭启风登基后，便将重霄追奉为帝君，并借重霄之名制订了一系列教规。关于他封神的传说便也是从这里开始的。因昭启风深受苍陆子民爱戴，又确是亲历过当年与沧烬间战争的英雄，他说的话，便成了人们心中毋庸置疑的事实。  
　　而紧接着没多久，颢宇陵谷一带又出现了上清宗。那师祖未冉师太自称受到了重霄帝君的感召，并受他指引行善救人传经诵道。她道法极高，济人无数，所传之道又给了战后无数迷惘的人们以希望，因此在颢宇有着极高的威望。这一东一西，看似毫无联系的地方不约而同地出现了同样地信仰，更是将重霄帝君的神话推向了一个无可撼动的高度。  
　　两派都认为帝君是司管人间的神，是天的意志的体现。  
　　云天宗主张世人不分尊卑贵贱，皆是天的子民。世间之所以有君臣之分，只因天赋与了人们不同的义务以及相应的权利。人与人之间应相互友爱。居上位者应善待一切天的子民，若只享了权利却没有付出相应的行动，便会受到天罚。相应的，居下位者应服从官府法制的管理。因为为官者的权利是天赋予的，相信他们便是相信天的选择。  
　　上清宗则淡化了所谓的地位之分，主张人虽生来有所不同，却只因承受有着不同的恩赐。贫穷与富贵都包含着不同的悲伤与快乐，而这些，都是人生宝贵的经历，都是上天的恩赐。人们要在一生中不断地通过将恩赐带给其他人，来偿还这一切。这样，才能够得到帝君的庇护，走过圆满的一生。  
　　看似有些不同，但实际只是角度的差别——不同地位所有的不同权利与义务、平穷于富贵总包含的悲伤与快乐，说到底乃是同一个东西。他们一个从国家出发，以维持秩序为本，讲大爱；一个从个人出发，以互助友爱为本，求小爱。昭启风和青妍，他们在世界的两端以各自不同的方式，将同样的思想传播着。  
　　可惜，随着时间的推移。或因误解，或因某些人的私欲，这两个教派却逐渐演变成了对等级制度的绝对保护与坚决抵制。直到三十多年前……  
　　“明明都自称是帝君的子民，到最后竟到了拼个你死我活的地步。”  
　　是的，那一年的慕蓉之战，慕军打着铲除异教的名号，将位于当时两国交界处的陵谷销毁殆尽，相传竟无一人幸存……  
　　“是啊，明明可以相互认可的事情，千百年来却总是争论不休。”我叹道，“明明渴望着相互的认同，却又本能地相互否认着。这便是关于人，我最弄不明白的地方。”  
　　不知昭启风和青妍若看到三十多年前的那一幕，又会作何感想？我在心里叹息着。  
　　  
　　夜色渐浓，我走在回颜府的路上，寻思着是不是过两天去钟离涣府上瞧瞧他可回来了没有——  
　　帝君祭前一天我起床后，钟离涣就已不见了踪影，只留下封信说漱愿之后不必等他，待他之后回到华京自会去找我。想必是调查有关沧烬的传闻去了。后来在庸州路上倒也确有听见三两的行人谈论起这个话题，还隐约听他们提起了什么“沧烬的肉身被找到了”，不过因是帝君祭前夕，这样的话题也就很快被带过了。  
　　如今，两个多月过去了，却仍旧没听到他有什么消息，偶问起他人，都说是蓉皇子身子向来虚，自五月初便去别处养病去了。  
　　养病的说法，我自然晓得是个幌子，可他的去处倒也的确无从打听。日子久了，便不由得渐渐担心起来，怕会不会是出了什么事……  
　　行至颜府大门口，却发现谁正立在门口的石狮子旁，见我来了，便自影中朝我走来……  
　　“许久不见，姑娘的漱愿可顺利完成了？”还未待看清钟离涣的脸，我就先听到了他略有些妖娆的声音。  
　　“托你的福，一切顺利。”  
　　“那便好。可有何感想？”  
　　“除了叹息，还能有什么……不过，后细细回想起来，又开始感叹这世人的可笑。”  
　　“哦？”  
　　“这三人间本有许多可歌可泣的东西，唯抛却了的那个，倒流传了下来，成了人们几百年来津津乐道的话题。”  
　　“惟留风月……姑娘你倒不妨这么想——那三人背负起了一生重如山的豪情与大义，特意将这柔似水的儿女情长留给了人间……如此算来，那‘可笑’的世人，不正是在帮他们填补此生的缺憾，续写着未曾来得及做过的梦吗？”  
　　钟离涣依旧是笑盈盈的，只是那眼神，似乎有些疲惫。  
　　  
　　“既是刚赶回来，何不明天再来寻我？偏挑这时。”钟离府的长亭中，我疑惑地望着他。  
　　“只因这两个月对姑娘朝思暮想，夜不能——”  
　　“好了好了，今夜就今夜罢。你这些话，留给舞坊的那些美人说去。”我有些不耐烦，真是搞不懂这个人，明知道我不信，却偏爱扯这些有的没的，“青妍的下落……寻着了么？”  
　　“嗯，果不出所料。”  
　　“未冉师太？”  
　　钟离涣看着我，点点头。  
　　“难怪人说她是个神仙，不分昼夜地济世救人，姿态又千变万化，时而如童真少女，时而又是个英姿飒爽的美人。”我回想起曾听过的那些传说，“原来其实是即墨浅与青妍二人。那些百姓们竟不曾怀疑这其实是二人所为！”  
　　“人们信愿意信的事，信大家都信的事情。姑娘不是最清楚这一点吗？”  
　　“只是不知即墨浅是怎么从沧烬教徒的身份，变回未冉那样的人物的。”  
　　“即墨浅与那‘沧烬’有些渊源。也是沧烬教众中少数幸存下来的人之一，据她本人说，是受了重霄帝君的什么恩。不过这些事，传了这几百年，孰真孰假，谁又分得清了。”  
　　“那么那把短剑呢？寻着了吗？”  
　　“已不在陵谷了。不过……”钟离涣神秘地笑笑，“几经走访，我已寻到它的下落了。”  
　　“那么你的打算是……？”  
　　“知道在谁那里就足够了。那个人，你也认识。现虽还在斐城，不过很快便要官复原职回到华京了……”  
　　许多条线索在斐城交汇，钟离涣所指的，竟是谦王世子昭辰——命运……真是奇妙。  
　　“不过，我倒是借来了这个。”他摊开掌心，露出一颗拇指盖大小的珍珠，“相传是未冉师太用来束发的珍珠，在她西归后便被在陵谷被供奉起来，后辗转被越宫收藏。虽不知是哪一位未冉的……”  
　　“想是青妍戴过的。”我仔细回忆着昨天的梦，青妍的发髻后头似乎是有这么颗珠子。  
　　“姑娘没兴趣探一探吗？”钟离涣笑道，搭在案上的手又朝我这边挪了挪。  
　　我一时又有些想问他是不是另有些什么企图，不过也罢，我也确实很想将那五百年前的旧事探个究竟。便覆上他的手，闭上了双眼……  
　　  
　　“姐姐真的不去吗？”即墨浅双臂撑在石桌上，十指交叉拖着下颌，问道。青妍坐在她对面的石凳上，遥望着远处。  
　　“还是你一个人去罢。”她淡淡地答道。  
　　“他会不会是察觉到什么，才邀我们去雁都讲道的？毕竟……重霄哥哥的事……”  
　　“也许罢，所以我更不能去了。待他见到了你，便再不会怀疑什么了。毕竟，谁又能想到未冉，其实是两个人呢。”  
　　“姐姐你……还在恨他吗？”  
　　“恨？”青妍仍旧望着远方，那眼神，像是忆起了什么往事似的，“这些年，我早想通了……或者说……其实他的缘由，我早就清楚，只是当时心里不想承认罢了。你还记不记得那时在庸州，你对我说：‘央已无可救药。’或许那时我就隐隐地明白了——人们要相信一件事情，总是需要些理由，即便这些理由并不代表什么。莫荏也曾说：‘同样的话，穷人说了就有人信，贵族说了就没人信。’国家也是一样，这便是古人所说的气数已尽罢……同一个昭启风，作为慕国的皇帝，他可以开创盛世，若换做央的皇帝，大概也只能与我父皇一样，用一生来拆东补西顾此失彼。”  
　　“唉……有些事，细想来很不理智，可这便是人，是人心，它们一点一点地汇聚起来，竟变成了那样可怕的力量，让人根本无从抗拒。而我……还有你的父皇，也曾在这力量面前迷失了自己——那时的我只是觉得：即便自己能够保持理智，又能如何？面对不可抗拒的人潮，只有让自己变得残忍，让自己脱离于他们，让自己驾驭他们——当时的我，竟被引入了这样的怪圈。”  
　　“这也怪不得你。相反，我之所以没有被引入同样的怪圈，与理智无关，或许，仅仅是因为一些反抗的本能罢。就好像我现在始终不愿意去面对启风一样……作为央的公主，作为戎的妹妹，作为父皇的女儿，我只是……没有办法，再去面对那一切……不过现在这样，也很好，不是吗？‘殊途同归’，我抛却公主、或是皇后的身份，和他朝着同样的方向努力着，终有一天，会在那尽头再遇的吧？”  
　　即墨浅没有说话，只是歪头细细打量着青妍，良久，“你知道吗？认识你的女孩子大多不喜欢你。我姐姐清也常对我说你极矫情的，让我不要被你骗了。这些话听得久了，后来也就不大和你来往了。”  
　　青妍尴尬的眨眨眼，不知说些什么好。  
　　“后来我自己也越发这么觉得了，”即墨浅继续说道，“我曾经很喜欢启风哥哥。偶有看到你们在一起，看到你有时娇蛮地朝他发脾气，看你偶尔做些‘傻事’，惹得他心疼怜惜。我就想起清姐姐说的话，心想真的是这样——更可恨的是，为什么上天，要给这样的人如此的面容和地位？”  
　　“那些娇蛮的事……我只是……希望和仅有的朋友，显得亲近些罢了。世人道我是公主，全都毕恭毕敬的，我只是想着做着那些明显是不懂事的事，说着明显是赌气的话的话，或许……他们会暂时忘了我是个公主，将我当成个普通的女孩子。可能，久而久之便习惯了……做得有些过火，但我从来都不想在你们面前做一个高高在上的公主。”  
　　即墨浅笑了起来，“你知不知道，当一个公主说出‘我也不想当公主’的时候，便是她最惹人讨厌的时候？”  
　　“我……知道……”青妍皱着眉，陷入了深深的矛盾。  
　　“后来我渐渐发现……其实在别人眼里，你做什么都是错——你不说话，是欲擒故纵；说话，是有心勾引；高高在上，是骄傲无情；平易近人，是虚情假意……我便开始思考：为什么？看那些史书、话本里的佳人的时候，明明和你差不多的，却是那么的喜欢，为何偏偏你就是错的？然后我发现，原来看书的时候，我想的是‘如若我是她’，而看着你，却时时刻刻的提醒着我‘我并不是她’。那一刻，我意识到了我自己是多么的丑陋……我意识到了世人又是多么地丑陋，还有那么多人，他们甚至都不知道自己是如何地丑陋。那时的我，无法忍受这样的丑陋……”即墨浅说这些时的语气，始终是冷冷的，像是在诉说着一个和自己并不相干的人一样。“后来，重霄哥哥唤醒了我。他让我知道了一点：先前我所认为的你的罪——你早已付出了代价，那便是他人的曲解与疏远、嫉妒与怀疑。那之后，我又把许多事都想了一遍，我才发现，真的如他所说……”  
　　青妍轻柔地摸摸她的头，像是在对她说：一切都过去了。“你们的话，曾对我打击很大。我久久地想不明白，为什么明明与我以往的认识差距如此之大的事情却能被你们说得那么合情合理？连一点反驳地余地都没有？那时，启风让我看远处的天地。后来，我才渐渐明白了。天地纵相隔万里，放到无限远的地方来看，也是相连的。善恶亦是如此。你们只是那站在遥远的地方看我们的人而已，站在那样的角度来看，善恶便无甚分别了。可我们自己终究是人，我们所看的天地该是我们自己的头顶与脚下。”  
　　“能在这样的年纪，把这样的道理翻来转去地悟了一通，究竟是幸还是不幸呢？”即墨浅幽幽地叹道。  
　　“和什么都没有想过的人相比，或许……要不幸一些罢。但和没有想得完整的人相比，我倒觉着是幸运了很多……”  
　　“在迷途的时候，被他们指引，便是我们所承的恩。”  
　　“所以，接下来，便轮到我们去向世人偿还这一切了。”  
　　即墨浅笑了起来，如寒冬过去，大地回春般，“那么姐姐若是不去的话。需要我向他说些什么吗？”  
　　“不用了，终有一天，他也会懂的。”  
　　  
　　“原来这便是未冉的来历……”我睁开眼，望着星空，感叹着。  
　　“唉……我的苍陆第一美人……”钟离涣一脸夸张的表情叹息着，惹得我好奇地打量着他。“还是怀念那个偶尔耍耍小脾气的公主殿下呐，不想竟变成了个看破红尘的道姑……苍天无眼呐！”  
　　“是吗？我倒觉得更加高洁美丽了……”  
　　“男人的心啊，你不懂的。你没听人说，女孩儿家，还是笨点的好。”钟离涣捂嘴笑着，“姑娘也要注意，凡事分析的太透彻了，可就再不招人喜欢了。”  
　　“自己看不穿，便看不得别人看得比自己透。”我有些不屑，“你们这些男人。”  
　　“姑娘此言差矣，”钟离涣的神情稍微正经了些，“关键的是：看得太透了，便少了些人情味，只能敬而远之了……”  
　　钟离涣似乎说得有些道理，只是我还不是很能体会。  
　　“我呀，就喜欢看那形形色色的人。但理性的那部分，‘因为所以’的，大家都差不多，多没意思。我就爱看剩下的那部分。”钟离涣眯眼看着我，“好比有姑娘闹别扭，便常有人感慨：‘明明我没做什么，这姑娘却为何生气了呢？’就是这一部分，不知为何的部分，才是人最妙的地方。各人各有自己的一番‘味道’，他人却学不来的。”  
　　“我倒也爱看那形形色色的人，只是……”  
　　“只是你只爱看那无趣的一部分。别说是你，即便是人，把这一部分看久了，也容易将自己的‘味道’给丢了。姑娘曾说自己想学着做人？其实你总想不通，然后干脆不去想的那一部分啊，才是人的精华所在。”  
　　我抿着嘴，似懂非懂地点点头，“算了，懒得听你一派胡言乱语……天色不早了，我该回去了。”说罢起身要走。  
　　“姑娘确定没有落下什么东西？”  
　　“嗯？”我疑惑地回过头去。  
　　“绕了半天，姑娘却把自己的初衷给忘了……”说着，他自怀中掏出一把墨玉笛，放入我的手中……
